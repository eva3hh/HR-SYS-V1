# Smart HR ERP System

## نظام إدارة الموارد البشرية الاحترافي

نظام ERP احترافي متكامل لإدارة الحضور والانصراف والموارد البشرية وحساب المرتبات.

## المواصفات الفنية

### Frontend
- React + TypeScript
- Material UI
- RTL Support (عربي)
- Dark & Light Mode
- Responsive Design

### Backend
- ASP.NET Core 9 Web API
- SQL Server Database
- Entity Framework Core (Code First)
- JWT Authentication
- Clean Architecture

## المميزات الرئيسية

1. **لوحة تحكم احترافية**
2. **إدارة الموظفين**
3. **نظام الحضور والانصراف**
4. **إدارة الشفتات والجداول**
5. **نظام الإجازات والأذونات**
6. **نظام حساب المرتبات**
7. **التقارير المتقدمة**
8. **نظام الصلاحيات**
9. **الإشعارات**
10. **سجل العمليات**
