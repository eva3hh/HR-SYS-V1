# Contributing Guide

## الإسهام في المشروع

شكراً لاهتمامك بالمساهمة في Smart HR ERP!

## خطوات المساهمة

1. **Fork المشروع**
```bash
git fork https://github.com/eva3hh/HR-SYS-V1.git
```

2. **إنشاء فرع جديد**
```bash
git checkout -b feature/your-feature-name
```

3. **قم بالتغييرات**
```bash
# عدّل الملفات حسب الحاجة
```

4. **Commit التغييرات**
```bash
git commit -am 'Add your feature'
```

5. **Push إلى الفرع**
```bash
git push origin feature/your-feature-name
```

6. **Open Pull Request**

## معايير الكود

- اتبع معايير C# و TypeScript
- أضف تعليقات واضحة
- اختبر التغييرات
- تأكد من عدم وجود أخطاء التجميع
